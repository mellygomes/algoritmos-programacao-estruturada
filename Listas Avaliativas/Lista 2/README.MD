##   Atividades avaliativas (prática com condições)

### Atividade 1:

* Receber dois valores do tipo **float**;
* Verificar qual é maior e qual é o menor;
* Imprimir.

### Atividade 2:

* Receber três valores do tipo **float**;
* Verificar e ordená-los do meior para o menor;
